# Code of Conduct — Group 31

## Academic Integrity

This capstone project is submitted as original work by Group 31 at Taylor's University.
All team members commit to:

- Submitting only **original code** authored by the team
- **Citing all references** used (papers, datasets, libraries)
- **Not plagiarizing** from other capstone projects or online sources
- **Acknowledging AI tools** if used for assistance (as per university policy)

## Team Collaboration Standards

### Respect & Professionalism
- Treat all team members with respect in all communication channels
- Constructive criticism only — no personal attacks
- Disagreements are resolved through discussion, not conflict

### Contribution Expectations
- Every member actively contributes to their assigned domain
- Absences must be communicated 24 hours in advance
- No one carries another member's workload without agreement

### Code & Commit Standards
- Follow the branching strategy in [CONTRIBUTING.md](CONTRIBUTING.md)
- Write meaningful commit messages (see CONTRIBUTING.md)
- Review teammates' pull requests within **48 hours**
- Do not merge your own PR — at least 1 team member must approve

### Communication
- Primary: **Google Chat** (daily async updates)
- Urgent: **WhatsApp**
- Meetings: Bi-weekly in-person + alternating Google Meet
- Supervisor: Weekly email progress reports

### Intellectual Property
- All code in this repository belongs to Group 31 and Taylor's University
- External libraries must be **open-source with compatible licenses**
- Dataset licenses (PlantVillage, PlantDoc) must be respected

## Enforcement

Violations of academic integrity will be escalated to:
**Supervisor:** Mr. Rabin Thapa | rabin@iimscollege.edu.np

Team-level conflicts will first be resolved internally, then escalated to the supervisor if unresolved within 3 days.
