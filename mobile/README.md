# Mobile — Flutter App

**Owner:** Jeshik Neupane (App Development Lead)

Cross-platform mobile application for plant disease diagnosis.

## Screens
- **Home** — Quick diagnosis button, recent history
- **Camera** — Capture leaf photo with guidance overlay
- **Diagnosis Result** — Disease name, confidence bar, treatment cards
- **History** — Past diagnoses
- **Login / Register** — JWT authentication

## Quick Start
```bash
flutter pub get
flutter run
```
